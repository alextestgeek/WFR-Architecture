# 09. Тест на проверку сложности и памяти (Memory & Complexity Test)

**Версия:** 0.1  
**Дата:** 30 марта 2026  
**Статус:** Планирование  
**Цель:** Экспериментально проверить декларации теории о сложности и памяти.

## Что проверяем

Теория утверждает:
- Память на токен: **O(1)** или **O(log n)**
- Сложность по длине контекста: значительно лучше O(n) или O(n²)

Мы должны это **измерить**, а не верить на слово.

## Что будем замерять

### 1. Память (Memory Usage)
- `torch.cuda.max_memory_allocated()` / `tracemalloc` (на CPU)
- Память на **один токен** при росте контекста
- График: Memory per token vs Context Length

### 2. Время (Wall-clock time)
- Время `forward()` для разных длин контекста
- График: Time vs Context Length
- Вычисление эмпирической сложности (O(n), O(log n), O(1))

### 3. Масштабирование
- Контексты: 512, 2048, 8192, 16384, 32768, 65536, 131072, 262144
- Сравнение с теоретическими O-нотациями

## Результаты теста (31 марта 2026, NVIDIA A100 80GB)

**Тест пройден успешно на GPU.**

**Ключевые результаты:**

| Context | Memory per token | Time (sec) | RC    | Active layers |
|---------|------------------|------------|-------|---------------|
| 512     | **492 bytes**    | 0.1424     | 0.9225| 2/4           |
| 2048    | **492 bytes**    | 0.0026     | 0.8176| 2/4           |
| 8192    | **492 bytes**    | 0.0029     | 0.7013| 2/4           |
| 16384   | **492 bytes**    | 0.0028     | 0.6448| 2/4           |
| 32768   | **492 bytes**    | 0.0029     | 0.5815| 2/4           |
| 65536   | **492 bytes**    | 0.0031     | 0.5198| 3/4           |
| **131072** | **492 bytes** | **0.0032** | 0.4607| 3/4           |

**Выводы:**
- Память на токен **строго постоянная** = **492 байта** (~0.48 KB) — **идеально соответствует O(1)**.
- Время forward после первого прогона ~**3 мс** даже на 131k токенов.
- GPU используется корректно (`CUDA: NVIDIA A100 80GB PCIe`).
- Resonance Confidence (RC) ожидаемо падает с ростом контекста.

**131072 токена** занимают **~64.5 МБ** видеопамяти (64487424 байт).

Это **сильный положительный результат** для теоретических утверждений.
