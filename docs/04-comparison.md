# 04. Сравнение WFR с другими архитектурами

## Сравнительная таблица (март 2026)

| Параметр                      | Transformer | Mamba-3 | MatMul-free | SNN     | **WFR**              |
|------------------------------|-------------|---------|-------------|---------|----------------------|
| Сложность контекста          | O(n²)       | O(n)    | O(n)        | O(n)    | **~O(n)** ¹          |
| Память на токен контекста    | O(n)        | O(1)    | O(n)        | O(n)    | **O(1)**             |

| Интерпретируемость           | Низкая      | Средняя | Средняя     | Низкая  | **Высокая**          |
| Бесконечный контекст         | Нет         | Частично| Нет         | Нет     | **Да**               |

## Ключевые преимущества WFR

- **Полный отказ от MatMul** — не просто оптимизация, а смена парадигмы
- **Физическая аналогия** — архитектура естественно ложится на волновые компьютеры
- **Масштабируемость контекста** — практически неограниченная
- **Event-driven природа** — вычисления происходят только при резонансе

¹ Теория предполагала O(log n), но эксперименты (до 100M токенов на A100) показали ~O(n). Память на токен подтверждена как O(1).
