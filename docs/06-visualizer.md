# 06. Интерактивный визуализатор

## Описание

В папке `src/` находится интерактивный визуализатор WFR-архитектуры, построенный на:

- React + TypeScript
- Three.js (3D волновое поле)
- Canvas 2D (визуализация фаз и спайков)
- Framer Motion (анимации)

## Что можно наблюдать

- **3D Resonant Field** — трёхмерное волновое поле, показывающее резонанс
- **2D Phase Interference** — стоячие волны и фазовая интерференция
- **Fractal Resonance Layers** — активность спайков по уровням фрактала
- **Live метрики** — loss, spike rate, амплитуды

## Как запустить

```bash
npm run dev
```

Откроется по адресу `http://localhost:5174`

## Назначение визуализатора

- Демонстрация принципов работы архитектуры
- Интерактивное исследование параметров (количество уровней, фаз, длины контекста)
- Визуальное доказательство концепции бесконечного контекста и event-driven поведения

Это не просто демо — это инструмент для понимания того, как будет работать будущая WFR-модель.
