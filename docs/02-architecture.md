# 02. Архитектура WFR

**Wave-Fractal-Resonant Architecture**

## 1. Основная концепция

WFR отказывается от парадигмы матричных умножений и линейных преобразований в пользу **волновой физики** и **резонанса**.

Информация в модели распространяется через **интерференцию фазовых волн** в фрактально организованной структуре.

## 2. Ключевые компоненты

### 2.1 Wave Phase Encoder (WPE)

**Цель:** Заменить позиционное кодирование и гипермерные вектора.

**Формула:**
```math
\phi_i^{(m)} = \left( 2\pi \cdot f_m \cdot i + \sum_{l=1}^{L} \alpha_l \sin(2\pi \beta_l \log_2(i+2) + \gamma_l) \right) \mod 2\pi
```

**Преимущества:**
- Память O(1) на токен (8–24 фазы)
- Иерархическое кодирование расстояний
- Отлично работает с фотонными чипами (фазовые модуляторы)

### 2.2 Fractal Resonance Layers

Многоуровневая фрактальная структура. Каждый уровень отвечает за свой масштаб паттернов.

**Резонансная функция:**
```math
\phi_l(z) = \sin(2\pi f_l z) \cdot e^{-\alpha |z-\theta_l|} \quad \text{при } |z| > \theta_l
```

### 2.3 Resonance-Triggered Spiking

Спайк генерируется только при **конструктивной интерференции**:

```math
R = \Re\left(\sum_j w_j \cdot e^{i(\phi_j - \phi_{target})}\right)
\text{spike} \iff |R| > \theta
```

Это делает систему **event-driven** и крайне энергоэффективной.

## 3. Механизмы стабильности (v2.1)

- **Phase-Locking (WPE-L)** — фазовая синхронизация каждые 4 частоты
- **Homeostatic regulation** — адаптивный порог спайка (целевая частота 10%)
- **Multi-scale surrogate gradient** — дифференцируемый спайкинг

Подробности и формулы: [03-theory.md](03-theory.md), разделы 7–8.

## 4. Поток данных

1. Вход → Wave Phase Encoder
2. Фазы + эмбеддинги → Fractal Resonance Layers (снизу вверх)
3. Активные резонансы вызывают спайки
4. Финальное состояние — **паттерн стоячих волн**

## 5. Обучение

Обучение в WFR-архитектуре реализуется через механизм **Resonant Field Plasticity (RFP)**, подробно описанный в [03-theory.md](03-theory.md). Архитектура поддерживает два режима обучения:
- Real-time (online) обучение
- Accelerated (offline) дообучение

Механизм основан на подстройке резонансных частот и использовании метрики **Resonance Confidence** для контроля распространения ошибки.

## 6. Аппаратная совместимость

- **Фотонная** — фазовые модуляторы и интерферометры
- **Нейроморфная** — спайковые чипы с осцилляторами
- **Квантовая** — phase gates и quantum phase estimation
