# WFR-Architecture

**Wave-Fractal-Resonant Architecture**

Принципиально новая архитектура ИИ, основанная на волновой физике, резонансе и фрактальной самоорганизации.

---

## Структура проекта

- **`docs/`** — вся документация и теория (начинать с `00-overview.md`)
- **`experiments/`** — научные эксперименты и тесты (основное направление работы)
- **`tools/`** — вспомогательные инструменты (включая визуализатор)
- **`research/`** — заметки, гипотезы, черновики
- **`RULES.md`** — строгие правила ведения проекта (обязательно к соблюдению)
