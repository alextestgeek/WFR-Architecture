# WFR-Architecture

**Wave-Fractal-Resonant Architecture**

Принципиально новая архитектура ИИ, основанная на волновой физике, резонансе и фрактальной самоорганизации.

---

## Актуальный статус проекта (30 марта 2026)

**Phase 0 (Viability & Stability) — завершён.**

### Что реально доказано:

- Базовая архитектура **жизнеспособна**. Smoke Test пройден.
- Система запускается, формирует стоячие волны, демонстрирует event-driven поведение.
- Мы можем получить **все 4 слоя активными одновременно** при правильном подборе параметров (особенно в `mean` и `frequency` режимах).
- `WavePhaseEncoder` работает корректно и соответствует формуле.
- Resonance Confidence (RC) ведёт себя ожидаемо — падает с ростом контекста.

### Что **не доказано** и остаётся открытым вопросом:

- **Масштабируемость** — слабая. При увеличении количества слоёв выше 6–8 большинство уровней становятся "мёртвыми". Это противоречит идее фрактальной иерархии.
- **Преимущества по сложности и памяти** (O(log n) или O(1) на токен) — **не подтверждены** экспериментально. Пока это только декларация.
- **Совместимость с фотоникой, нейроморфными и квантовыми системами** — только теоретические предположения. Нет ни одной симуляции.
- **Механизм обучения (RFP)** — полностью отсутствует. Мы тестировали только forward pass необученной системы.
- **Сравнение с существующими архитектурами** — не проводилось.

### Текущие рекомендации

- Оптимальное количество фрактальных уровней на текущий момент: **4–6**.
- Лучшие найденные конфигурации зафиксированы в [`docs/strategy-fractal-layers.md`](docs/strategy-fractal-layers.md).

**Следующий этап:** Переход к **Phase 1** — реализации базового механизма обучения **Resonant Field Plasticity (RFP)**.
