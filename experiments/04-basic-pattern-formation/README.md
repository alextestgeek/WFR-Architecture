# Experiment 04 — Basic Pattern Formation (Test 2, Phase 0)

**Дата:** 31 марта 2026  
**Версия архитектуры:** v2.1  
**Статус:** Завершён (прогон локально CPU; GPU опционален)

## Связь с теорией

Проверяется [`docs/03-theory.md`](../../docs/03-theory.md) (раздел 5) — стоячие волны как носитель структуры: разные **порядки позиций** (синтетические паттерны одной длины) должны давать **различимые** агрегированные представления без обучения.

## Гипотеза

При фиксированных весах и замороженном homeostatic после warmup разные классы позиционных последовательностей дают попарно различные сигнатуры (профиль стоячей волны по четвертям + спайки + RC). Тот же паттерн даёт ту же сигнатуру (детерминизм).

## Метод

- **Модель:** `WFRNetwork`, конфигурация Freq-Balanced (как в Phase 0).
- **Warmup:** линейный `0..L-1`, 50 проходов; затем `homeostatic_enabled = False`.
- **Длина:** `L = 4096`.
- **Классы паттернов** (все `positions` в диапазоне `0..L−1`, int64, как вход WPE / `WFRNetwork`):
  - `linear` — 0, 1, …, L−1  
  - `reverse` — L−1, …, 0  
  - `mod64` — `i % 64`  
  - `stride7_mod_L` — `(i * 7) % L`  
  - `shuffle_seed12345` — перестановка множества {0,…,L−1}  
  - `two_blocks` — два соседних полублока с одинаковой половиной индексов  

- **Сигнатура:** L2-нормированный вектор из: средних стоячей волны по **4 четвертям** последовательности, `std(sw)`, средних спайков по 4 слоям, RC (размерность 10).

## Этапы

| Этап | Проверка | Критерий PASS |
|------|----------|----------------|
| A | Различимость классов | max off-diagonal cosine **< 0.99999** |
| B | Повторяемость | cosine сигнатур двух прогонов `linear` **> 1 − 1e−5** |
| C (опц.) | Шум по фазам после WPE (`σ=0.02`) | cosine(clean, noisy) **> 0.85** |

**STRONG:** max off-diagonal cosine < 0.995 (более строгая попарная различимость; при необученных весах выполняется не для всех пар).

**Интерпретация:** PASS по этапу A означает, что **ни одна** пара классов не даёт почти совпадающих L2-нормированных сигнатур (макс. попарный косинус < 0.99999). Это слабый критерий: векторы могут оставаться почти коллинеарными. На референсном прогоне запас до порога 0.99999 у пары `stride7_mod_L` / `shuffle_seed12345` небольшой. Класс `mod64` сильно отличается от остальных и вносит основной вклад в низкий min off-diagonal cos.

## Результаты (прогон 2026-03-31, CPU)

| Метрика | Значение |
|---------|----------|
| max off-diagonal cos | 0.999934 |
| min off-diagonal cos | 0.897 |
| repeatability (linear) | 1.0 |
| cos(clean, noisy phases) | ~0.99998 |

- **Различимость (A):** PASS (порог 0.99999).  
- **Повторяемость (B):** PASS.  
- **Шум (C):** PASS по порогу 0.85.  
- **STRONG:** не достигнут (max cos > 0.995).

**Вердикт:** **Подтверждено** (критерии A+B в мастер-плане; необученная сеть оставляет близкие пары `stride`/`shuffle`).
