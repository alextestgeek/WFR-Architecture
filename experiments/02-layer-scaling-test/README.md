# Experiment 02 — Layer Scaling Test

**Дата:** 31 марта 2026  
**Статус:** Завершён  
**Платформа:** NVIDIA A100 80GB

## Гипотеза

Механизмы стабилизации v2.0 (Phase-Locking, Homeostatic, Surrogate Gradient) позволяют масштабировать WFR-архитектуру до 32 фрактальных слоёв без деградации.

## Критическая находка

Обнаружен **баг инвертированного знака** в Homeostatic regulation:

```python
# БАГ (v2.0): delta = η * (r_target - r_real) — положительная обратная связь
# ИСПРАВЛЕНИЕ (v2.1): delta = η * (r_real - r_target) — отрицательная обратная связь
```

Это была единственная причина "мёртвых" слоёв. После исправления — 32/32 слоёв активны.

## Параметры теста

- **Глубины:** 4, 8, 16, 24, 32 слоя
- **Контексты:** 512, 8 192, 131 072 токенов
- **Стратегии частот:** linear, logarithmic, harmonic, constant
- **Homeostatic warmup:** 200 forward passes
- **Начальный порог:** θ₀ = 0.3 для всех слоёв
